# Icon 图标

----- IconName

## `name`?: React.ReactNode

名称

----- IconColor

## `color`?: 'normal' | 'gray' | 'blue' | 'red' | 'orange' | 'green' | string

颜色

----- IconSpinning

## `actions`?: boolean = false

是否旋转

----- IconFit

## `fit`?: boolean = false

是否间隙

----- IconClickable

## `clickable`?: boolean = false

是否可点击

----- IconOnClick

## `onClick`?: React.MouseEventHandler

点击事件
